Huawei Agentic Cloud Console

AI & Cloud | Innovation & PoC

Defining future patterns in agentic cloud era | 2.5x Faster UX Delivery with AI-Accelerated Workflow

This project focuses on Huawei Cloud Console, the core portal for enterprise users to manage cloud resources. As agentic capability grow, we see the fundamental restructure of cloud interface with cloud architecture the potential pivot point for Huawei Cloud to surpass competitors.

Problem:

  1. As customers grew in scale on HWC, they find the platform complicated to use for daily operation and monitoring. Services are getting better and diversed, but existing / classic console interfaces, even with AI, are static, passive, information overwhelming and still a lot of manual effort / attention needed. Current conversational UI for AI is just not sufficient.
  2. In a future setting of agentic cloud, customers will meet amplified challenges of using and trusting the system for cloud domain. Even when the agentic level is L4~5, human will find it speculative and uncertain to work with.

Constraints:

  • Uncertain and progressing of future cloud architecture
  • US sanctions on tooling options (no access to leading design and AI tools)

Strategy: Human-like Collaboration

Establish overall human like collaborator mental modal (autonomous + ambient in the loop).

Principles:

  • Beyond Conversation UI:
    • Shift to cognitive experience in cloud use case.
    • Project management like canvas UI for conversation navigation pattern.
  • Generative Dashboard: Mix of generative UI with fixed UI.
  • Collaborative: Real time collaboration with agent and self healing.
  • Omnipresent Integration: Integration into daily tools as ambient agent standby.
  • AI Adoption in UX Process:
    • Went in a very narrow list and find Tempo.ai as the vibe designing tool
    • Use vibe coded prototypes in concept testing phase, and complete 7 prototypes in 2 days

Prototypes built with Tempo.ai

2.2 App building

Impact

  • Lock implementation for key pattern.
  • Embedded generative AI into the design pipeline, increasing prototyping throughput 2.5× — accelerating production from 7 prototypes in 5 days to 7 in 2 days — and significantly shortening cross-functional decision cycles. Defined and drove cross-organizational vision for AI-native cloud experience, shaping the roadmap toward secure, agentic operational models and aligning design, engineering, and product leadership around next-generation intelligent control-plane strategy.
