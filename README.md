# Portfolio Website

A minimal, responsive portfolio website showcasing design work and projects.

## Features

- **Responsive Design**: Optimized for mobile, tablet, and desktop (375px - 1440px+)
- **Modern UI/UX**: Following Web Interface Guidelines and 8px grid system
- **Sticky Footer**: Clean layout with proper vertical rhythm
- **Motion-Driven**: Smooth transitions and hover states
- **Accessible**: Proper contrast ratios and semantic HTML

## Pages

- **Home** (`index-v2.html`): Hero section with project list
- **About** (`about-v2.html`): Personal information and experience
- **Playground** (`playground-v2.html`): Creative projects showcase
- **Work Detail** (`work-detail-v2.html`): Individual project details
- **Resume** (`resume.html`): Professional resume

## Tech Stack

- HTML5
- CSS3 (Vanilla CSS with custom properties)
- Google Fonts (Outfit, Roboto, Inter)
- SVG Icons

## Design System

- **Colors**: Monochrome palette with blue accent (#2563EB)
- **Typography**: Outfit (headings), Roboto (body), Inter (UI)
- **Spacing**: 8px grid system (20px, 64px, 80px)
- **Breakpoints**: 
  - Mobile: ≤768px
  - Tablet: ≤1024px
  - Desktop: ≥1025px

## Local Development

Simply open any HTML file in your browser:

```bash
open index-v2.html
```

## Deployment

This is a static site and can be deployed to:
- GitHub Pages
- Vercel
- Netlify
- Any static hosting service
